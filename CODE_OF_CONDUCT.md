# Code of Conduct

## Our pledge

Be respectful. Assume good intent. No harassment, hate, or personal attacks.

## Scope

Applies to project spaces (issues, PRs, discussions) and representation of the project.

## Enforcement

Report abuse via GitHub security/private contact on the repo. Maintainers may warn, moderate, or ban.
